# Usage

Run the TUI:

```
python -m tz_player.app
```

Current keys:
- `j` / `k`: move cursor down/up in the playlist
- `shift+up` / `shift+down`: reorder selection up/down
- `s`: toggle selection for the current row
- `f`: focus the Find input
- `space`: play/pause placeholder (logs "not implemented")
- `escape`: dismiss modals
